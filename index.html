About This Application

This website is a simple single page application designed for browsing and viewing clothing products online. The goal was to create a clean interface where users can explore items, open a detailed view for each product, and manage a small shopping cart.

The application was built using:

  • Visual Studio for development
  • Tailwind CSS for styling via CDN (There is no external CSS file).
  • JavaScript techniques learned throughout the labs and tutorials in this course. Many of the functions and approaches in the JavaScript file are based on lab activities, customized to support the specific functionality required for this assignment.
